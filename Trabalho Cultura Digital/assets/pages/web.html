Web

o que é web?

A segurança web protege informações e sistemas online contra acessos não autorizados, a web é uma plataforma composta por bilhões de páginas interconectadas por links,Os usuários acessam a web através de navegadores e interagem com conteúdo em HTML, CSS e JavaScript, por exemplo, esse site.

Além das páginas estáticas, a web suporta aplicativos interativos. Resumindo, a web é uma plataforma global que facilita comunicação, comércio e compartilhamento de conhecimento, com medidas de segurança para proteger usuários e dados.

tipos de web

Existem diferentes tipos de web, que variam em termos de funcionalidade, acesso, conteúdo e propósito. Os três tipos de web: Surface Web, Deep Web, Dark Web.

Surface Web

A World Wide Web, também chamada de internet ou Surface Web, é a parte acessível diariamente por meio de navegadores tradicionais. Ela inclui sites de tecnologia, serviços essenciais, lojas online, redes sociais e canais de entretenimento.

Deep Web:

A deep web é uma parte da internet não acessível pelos mecanismos de busca comuns, contendo informações confidenciais e privadas. Inclui registros médicos, dados acadêmicos e outros conteúdos legítimos que demandam maior segurança. Assim como os bastidores de um programa de TV, é uma área invisível onde ocorrem processos importantes.

Dark Web:

A Dark Web é uma camada abaixo da Deep Web, acessível através de um navegador específico chamado Tor. Nessa área, há menos supervisão e rastreamento, o que favorece práticas criminosas. É conhecida por oferecer alta privacidade e dificuldade de rastreamento de atividades. Se desejar explorá-la, é necessário usar software de criptografia para proteger seus dados, pois todos os usuários estão vulneráveis a ataques cibernéticos.

Acessar A Dark Web ou a Deep Web é Crime?

Acessar a deep web não é ilegal em si, mas realizar atividades ilegais por meio desse acesso pode ser punível por lei. A deep web contém conteúdo além do ilegal, sendo útil para ativistas e jornalistas que buscam anonimato. O navegador Tor é comumente usado para acessar a deep web.

Na Dark Web, é importante agir dentro dos limites legais, pois há riscos de malware, golpes e associação involuntária com atividades ilegais, participar de atividades criminosas na Dark Web pode resultar em consequências legais, como investigação e prisão. É essencial agir em conformidade com a lei ao navegar na internet, inclusive na Dark Web.

Como entrar na deep web:

Você pode acessar a deep web usando seu smartphone Android ou iOS, mas é necessário instalar aplicativos específicos para estabelecer a conexão. Aqui está um resumo de como acessar a deep web no celular e quais apps são necessários para isso.

Primeira Etapa:

Baixe e instale o Tor Browser, um navegador específico para acessar a deep web.

Segunda Etapa:

Conecte-se à rede permitindo que o navegador se conecte.

Teceira Etapa:

Escolha o nível de segurança adequado, preferencialmente o recomendado para usuários inexperientes.

Quarta Etapa:

Uma vez que o navegador esteja em funcionamento, digite uma URL válida para acessar sites na deep web e começar a navegar.

Curiosidades Sobre Web:

A web evoluiu consideravelmente desde sua criação, tornando-se uma plataforma global de grande impacto. Suas curiosidades ressaltam marcos significativos, mudanças na maneira como as pessoas a utilizam e interagem, refletindo seu papel cada vez mais relevante na sociedade.

Não por acaso, a metáfora mais utilizada para ilustrar a web é a do iceberg: nela a Surface Web é apenas a área aparente, que fica acima do nível do mar. Abaixo dela está a Deep Web, e dentro desta, a Dark Web.

Deep Web e Dark Web não são a mesma coisa Muita gente confunde as duas coisas, mas é importante saber que elas são diferentes. Diferente da Deep Web, a Dark Web é uma pequena fatia desse universo. É lá que estão concentrados os materiais, informações e vendas de cunho criminoso.

Quem escuta falar em Deep Web normalmente também escuta falar sobre vídeos Snuffs. Esses vídeos são, provavelmente, o que há de mais bizarro em toda a internet. Isso porque seu conteúdo traz gravações de morte, violência, tortura e tudo o que há de mais brutal e doentio. Tem louco para tudo, né?

A Bitcoin era a moeda original preferida, mas vem perdendo esse status porque a polícia já consegue rastrear algumas transações feitas com ela. Os administradores do mercado da Dark Web estão experimentando outras moedas, como Monero e Litecoin.
