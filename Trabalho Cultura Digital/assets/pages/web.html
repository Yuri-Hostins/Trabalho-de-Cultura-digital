Web

o que é web?

A web, ou World Wide Web, é um sistema de informações interconectadas que permite o acesso a conteúdos como documentos, imagens, vídeos e outros recursos multimídia por meio da internet. Foi criada por Tim Berners-Lee em 1989 e se tornou uma das aplicações mais populares da internet.

A web funciona com base em uma arquitetura cliente-servidor, onde os usuários utilizam um programa chamado navegador para acessar os recursos disponíveis na web.

A web utiliza uma linguagem de marcação chamada HTML (Hypertext Markup Language) para estruturar e organizar o conteúdo das páginas. Além disso, são utilizadas outras tecnologias como CSS (Cascading Style Sheets) para definir a aparência visual das páginas, e JavaScript para adicionar interatividade e funcionalidades dinâmicas.

A web é composta por bilhões de páginas e sites, que são interligados por meio de links. Os links permitem que os usuários naveguem de uma página para outra, tornando a web uma rede global de informações interconectadas.

Além das páginas estáticas, a web também suporta aplicações web interativas, onde os usuários podem realizar tarefas complexas, como fazer compras online, usar redes sociais, assistir a vídeos em streaming, entre outros.

Em resumo, a web é uma plataforma que permite o compartilhamento e acesso a uma ampla gama de informações e serviços online, facilitando a comunicação, o comércio e o compartilhamento de conhecimento em escala global.

tipos de web

Existem diferentes tipos de web, que variam em termos de funcionalidade, acesso, conteúdo e propósito.
Os três tipos de web: Surface Web, Deep Web, Dark Web.

Surface Web

É a parte que acessamos todos os dias, em que encontramos sites de tecnologia (como o Tecnoblog e o Meio Bit), serviços essenciais, lojas eletrônicas, redes sociais ou canais de entretenimento, é a World Wide Web ou internet, também conhecida como Surface Web.

Exemplos: todo o seu conteúdo é acessível por meio de navegadores tradicionais como Google Chrome, Safari ou Firefox.

Deep Web

A deep web, traduzindo para o Português, seria a web escondida, oculta ou profunda. Ela possui uma quantidade baixa de regulamentação. No caso, você não pode simplesmente pesquisar seu conteúdo no Google ou similar, por exemplo, pois ele não será encontrado por esses motores de busca. No entanto, ela não necessariamente é associada a ações clandestinas.

Vamos imaginar que você deseja consultar um diagnóstico de uma consulta médica que você realizou em um hospital. Isso seria um exemplo em que a deep web se encaixa, pois, os registros médicos encontram-se nessa parte da web. Não só eles, como outras informações sigilosas acadêmicas, financeiras, enfim, tudo que envolva um nível de confidencialidade nacional.

A deep web, seria uma espécie de parte oculta na internet que podemos associar aos bastidores de um programa de TV. Para todo o programa televisivo ser transmitido, ele precisa de vários ajustes de fios, satélites, etc. E, a parte dos bastidores é exibida na TV? Em grande parte dos casos, não. Ou seja, seria a parte “oculta” dos programas e, isso é similar ao que ocorre na internet.

Acessar a deep web é crime?

Acessar a deep web não é considerado crime em si, apenas ações ilegais que possam ser realizadas por meio desse acesso é que são passíveis de punição pela lei”, explica Miguel Pereira Neto, especialista em direito criminal, sócio do Lacaz Martins, Pereira Neto, Gurevich & Schoueri Advogados.

O que fazer na deep web,Geralmente, a deep web é associada a conteúdo ilegal, como venda de drogas, pornografia infantil ou diversas outras atividades obscuras, mas a rede não tem só esse tipo de conteúdo.

Ela pode ser bastante útil para ativistas e jornalistas que precisam de anonimato. No computador, ela pode ser acessada pelo navegador Tor, que se conecta à rede de mesmo nome.

Normalmente, o primeiro passo é acessar a Hidden Wiki, que lista alguns sites da deep web. O diretório mais famoso do Tor é a Hidden Wiki. Por lá, você encontra qualquer tipo de site que existe nas redes da deep web.

Como entrar na deep web

Acessar a deep web é algo que também pode ser feito pelo smartphone, seja Android ou iOS. Da mesma forma que no computador, é preciso instalar alguns aplicativos de modo que a conexão seja feita no seu smartphone. Por isso, a seguir, você confere como entrar na deep web pelo celular e quais apps precisam seu baixados.

  • Baixe e instale o app Primeiramente, vá até a Play Store, faça o download do Tor Browser e instale-o em seu dispositivo;
  • Faça a conexão Assim que instalado, permita que o navegador conecte-se à rede. Assim, será possível acessar as URLs próprias da deep web;
  • Selecione o nível de segurança Em determinado momento, será necessário escolher o nível de segurança que deseja ter. Deixe no recomendado caso não tenha experiência no assunto;
  • Acesse à rede Pronto, navegador em funcionamento. Agora, é preciso que você acesse uma URL válida para começar a navegar pela deep web.

Dark Web

A Dark Web seria um nível ainda abaixo da Deep Web e, para ela, precisamos ter um navegador específico para acessar, chamado Tor. Aqui, a fama de algo oculto e clandestino começa a fazer sentido, pois, como há ainda menos supervisão e possibilidade de rastreamento, muitas práticas criminosas surgem nessa camada.

Portanto, a Dark Web fornece um alto nível de privacidade e, com isso, fica quase impossível rastrear as ações que são realizadas por pessoas. Caso você deseje se aventurar na Dark Web, deverá utilizar softwares poderosos de criptografia para proteção de seus dados, pois todos que estão nessa rede são vulneráveis a todos os tipos de ataques cibernéticos que possam ocorrer.

Acessar A Dark Web É Crime?

Entrar na Dark Web não é um crime em si, desde que você esteja apenas acessando-a para fins legítimos e não esteja participando de atividades ilegais enquanto estiver lá. Por exemplo, pesquisar e explorar a Dark Web por curiosidade ou para entender melhor o seu funcionamento não é ilegal.

No entanto, é importante ter cuidado ao acessar a Dark Web, pois ela é conhecida por ser um ambiente perigoso e propício a atividades criminosas. Há riscos de malware, golpes, roubo de identidade e até mesmo a possibilidade de ser associado a conteúdos ilegais sem intenção.

Em muitos países, participar de atividades ilegais na Dark Web é considerado um crime. Se você se envolver em atividades criminosas, como comprar ou vender drogas ilegais, armas ilegais ou conteúdo ilegal, poderá enfrentar consequências legais, incluindo investigação, acusação e possível prisão.

Portanto, é sempre recomendado agir dentro dos limites legais ao navegar na internet, incluindo a Dark Web, e evitar qualquer atividade que viole a lei.

Curiosidades Sobre Web

Essas curiosidades destacam o desenvolvimento e a evolução da web ao longo dos anos, desde sua invenção até sua expansão global e as mudanças na forma como as pessoas acessam e interagem com ela.

Não por acaso, a metáfora mais utilizada para ilustrar a web é a do iceberg: nela a Surface Web é apenas a área aparente, que fica acima do nível do mar. Abaixo dela está a Deep Web, e dentro desta, a Dark Web.

Deep Web e Dark Web não são a mesma coisa Muita gente confunde as duas coisas, mas é importante saber que elas são diferentes. Diferente da Deep Web, a Dark Web é uma pequena fatia desse universo. É lá que estão concentrados os materiais, informações e vendas de cunho criminoso.

Quem escuta falar em Deep Web normalmente também escuta falar sobre vídeos Snuffs. Esses vídeos são, provavelmente, o que há de mais bizarro em toda a internet. Isso porque seu conteúdo traz gravações de morte, violência, tortura e tudo o que há de mais brutal e doentio. Tem louco para tudo, né?
